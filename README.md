# MiniHack: AI Data Intelligence Challenge

## Problem Statement
Analyze a last-mile delivery dataset to identify the major causes of delivery delays using Python, Data Analysis, EDA, Statistics, and Machine Learning.

## Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- SciPy

## Project Workflow
- Data Loading
- Data Cleaning
- Missing Value Handling
- Feature Engineering
- Exploratory Data Analysis
- Statistical Testing
- Dashboard Creation
- Business Insights

## Key Findings

### Q1
- Peak Hour Mean Delay = 23.25 mins
- Off-Peak Mean Delay = 7.38 mins
- Difference = 15.86 mins

### Q2
- Fog Median Delay = 37.9 mins
- Rain Median Delay = 29.4 mins
- Medicine orders experienced the highest delay during rain.

### Q3
- Under 2 Years Mean Delay = 13.75 mins
- Over 4 Years Mean Delay = 14.35 mins
- No statistically significant difference (p = 0.631).

### Q4
Operational Recommendation:
Increase rider availability and optimize routing during peak hours (8–10 AM and 5–8 PM).
